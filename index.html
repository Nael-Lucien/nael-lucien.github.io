02 — Projets

Ce que j'ai construit

// PROJECT_01

Arcane Hybrid Worlds

Prototype de monde ouvert réalisé en stage à Robert Gordon University (Aberdeen, Écosse) dans le cadre d'un projet international. Univers inspiré de la série Netflix Arcane — conception des environnements 3D sous Blender, scripting des mécaniques de jeu en C# et intégration sous Unity. Comprend la création d'un musée virtuel interactif dédié aux éléments du lore d'Arcane, avec plusieurs mini-jeux immersifs permettant aux joueurs d'explorer l'univers de la série.

UnityC#Blender

// PROJECT_02

TransChlorWeb

Portage web d'un simulateur scientifique de transport d'ions chlorure, initialement développé en Visual Basic. Migration complète vers C# / Symfony 7, avec refonte de l'interface et exposition des calculs physico-chimiques via une API REST. Projet mené en collaboration directe avec des chercheurs, nécessitant une compréhension approfondie des modèles mathématiques sous-jacents. Déployé sous Docker, avec base de données SQL et interface moderne en Tailwind.

Symfony 7PHPTailwindDockerSQLAPIVisual BasicC#

// PROJECT_03

Jacquard — Musée des Soieries

Application web pédagogique développée pour le Musée des Soieries Bonnet (Jujurieux, Ain) dans le cadre d'une SAE en méthodologie Scrum. Modélisation 3D complète du métier à tisser Jacquard sous Blender, intégrée en temps réel via Three.js / TypeScript. Propose un mode guide interactif permettant aux visiteurs de comprendre le fonctionnement du métier, ainsi qu'un mini-jeu d'initiation au tissage. Backend PHP avec SQLite, conteneurisé sous Docker pour le déploiement sur site au musée.

TypeScriptPHPBlenderViteSQLiteDocker

// PROJECT_04

GASC — Gestion Activités

Application de gestion d'activités culturelles et sportives développée en équipe sous Symfony 6. Intègre un système d'authentification multi-rôles (admin, organisateur, participant), un module de planification et d'inscription aux événements, ainsi qu'un tableau de bord administrateur avec statistiques. Interface réalisée avec Bootstrap, persistance des données en SQL via Doctrine.

Symfony 6PHPBootstrapSQL
